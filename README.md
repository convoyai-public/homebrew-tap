# homebrew-tap

Homebrew tap for Convoy AI internal tools.

## Usage

```
brew install convoyai-public/tap/cwf
```

Formulae in this tap are published automatically by each tool's release
pipeline (see `cwf/.goreleaser.yaml` in
[eng-dev-tooling](https://github.com/convoyai/eng-dev-tooling)). Don't
hand-edit files under `Formula/` — they're regenerated on every tagged release.
